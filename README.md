# Development
## Requirements
- Python 3.7+
- PyInstaller 4.1

`UMvC3_Stage_Installer.exe` code resides in `installer.pyw`

`umvc3stg_file_generator.exe` code resides in `umvc3stg_generator.pyw`

After making changes, run `build.bat`
